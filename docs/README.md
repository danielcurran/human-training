## How This Repo Learns

We collect feedback from every lab run:
- What was clear? What was confusing?
- Where did learners get stuck? For how long?
- What prior knowledge do learners have?
- What are the most common mistakes?

That feedback feeds back into lab iteration and standards refinement.
