# Skills

Reusable domain knowledge modules and teaching strategies for MongoDB education.

## Structure

Each skill is a folder containing:
- `SKILL.md` — The skill definition and guidance
- Optional: `examples/` — Worked examples and case studies

## Creating a New Skill

1. Create a folder: `skills/{skill-name}/`
2. Write `SKILL.md` with:
   - Purpose and scope
   - When to use this skill
   - Key principles
   - Examples or templates
3. Reference it in lab specs and outlines
